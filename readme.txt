Adds an API allowing to store and query term meta as it is done with post meta.

== Description ==

Adds an API allowing to store and query term meta as it is done with post meta using the next functions:

* `add_term_meta`
* `update_term_meta`
* `delete_term_meta`
* `get_term_meta`

Also adds `meta_compare` option to get_terms call.

== Installation ==

1. Upload the entire `termmeta-api` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Use the API
